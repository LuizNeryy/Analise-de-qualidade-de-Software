# Stack Técnico

## Frontend

- Ruby on Rails com Spree no storefront principal.

## Backend

- Flask e SQLAlchemy para os microserviços de anúncios e descontos.
- ActiveRecord e controllers do Spree/Rails para o storefront.

## Dados

- PostgreSQL 11-alpine nos serviços Flask.
- SQLite no estado inicial do aplicativo Rails.

## Observabilidade

- Datadog Agent.
- Datadog APM.
- Datadog Browser RUM.
- Logs com enriquecimento por serviço.
- Service Catalog.
- Synthetics.

## Infraestrutura

- Docker e Docker Compose.
- Kubernetes.
- Nginx.
- Terraform.
- Suporte a AWS, Azure, GCP e OpenShift.

## Ferramentas auxiliares

- GoReplay.
- Scripts de build para múltiplas imagens.
- Configurações de CI e release em GitHub Actions.
