# Funcionalidades do Sistema

## Storefront principal

- Navegação de produtos.
- Visualização de detalhes de produto.
- Fluxo de checkout.
- Integração com anúncios no detalhe de produto.
- Integração com o serviço de descontos.

## Serviço de descontos

- Listagem de descontos via endpoint HTTP.
- Criação de novos descontos.
- Serialização de desconto com tipo associado.
- Exibição de dados relacionados a influenciadores.

## Serviço de anúncios

- Listagem de anúncios.
- Entrega de imagens de banner.
- Seleção de banners por peso.
- Criação de novos anúncios.

## Frontend auxiliar em React

- Consulta de descontos externos.
- Ordenação por nome, código e valor.
- Filtro de descontos salvos.
- Salvamento local no navegador.
- Troca manual de anúncios.
- Instrumentação de RUM e ações customizadas.

## Infraestrutura e operação

- Replay de tráfego com GoReplay.
- Instrumentação e coleta de telemetria pela Datadog.
- Definições de serviços no Service Catalog.
- Variações de deploy para comparação de versões.
